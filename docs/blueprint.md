# **App Name**: UXFolio

## Core Features:

- Blank Page Display: Display a blank page with 'Hello' at the center.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to represent professionalism and sophistication.
- Background color: Very light desaturated Indigo (#E8EAF6) for a clean, unobtrusive backdrop.
- Accent color: Violet (#7953D2) to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral feel.
- Center the 'Hello' text using flexbox for precise alignment.
